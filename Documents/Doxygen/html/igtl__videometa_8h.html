((n) * IGTL_VIDEOMETA_ELEMENT_SIZE)

igtl_videometa_get_data_size(n) macro calculates the size of body based on the number of images.The size of body is used in the message header. igtl_videometa_get_data_n(size) calculates the number of images in the body, based on the body size. This function may be used when a client program parses VIDEOMETA message.
